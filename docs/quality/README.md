# Quality Score History

`scripts/record-quality-score.sh` writes commit-scoped quality score reports here.

- `<commit>.json`: machine-readable score and metrics
- `<commit>.md`: human-readable score summary
- `latest.json` / `latest.md`: latest recorded run

The score is calculated from a fixed Markdown fixture generated by
`examples/quality_score.rs`. The acceptance gate remains `scripts/verify.sh`;
the score is an additional trend signal for generated wiki output quality.
