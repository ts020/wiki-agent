# 5. ユースケース

## UC-01 単発メモの wiki 化
1 枚のメモを試しに入れて wiki の形を確認する。

```sh
md-wiki init path/to/note.md
```

→ `./md-wiki/notes/note.md` と最小限の索引が生成される。

## UC-02 既存メモ群を一括 wiki 化
`~/notes` 配下を全部取り込んで自分用 wiki を作る。

```sh
md-wiki init ~/notes -o ~/my-wiki
```

## UC-03 増築
既存の wiki に後から md を追加する。新規 md をディレクトリに置いて、`md-wiki add -o ~/my-wiki` を実行すると索引・バックリンク・未解決リンクの解決が更新される。

## UC-04 タグ横断
`tags/auth.md` を開くと、`auth` タグを持つすべてのノートが列挙されている。ネストタグ `auth/session` は `tags/auth.md` と `tags/auth/session.md` の両方から到達できる。

## UC-05 見出し横断
`headings/` から「全ノートのどこに何という見出しがあるか」を一覧で眺め、興味ある章に直接ジャンプする。

## UC-06 リンク関係の可視化
`links/` を開くと「どのノートがどのノートを参照しているか」を一覧で確認できる。各ノートにはバックリンク欄も自動付与される。

## UC-07 未解決リンクの確認
`_unresolved.md` で「まだ実体の無い `[[NewTopic]]`」を把握し、次に書くべきノートを判断する。
