# 15. 確定済み仕様

v1.0 策定時点で判断が固まった項目。詳細は対応する FR / 節を参照。

| 項目 | 結論 | 参照 |
|---|---|---|
| ツール名 | `md-wiki` | [§1.1](01-概要.md#11-名称) |
| 対応入力 | `.md` のみ。コードは対象外 | [§2](02-スコープ.md) |
| 入力形式 | 単一ファイル（既定）/ ディレクトリ + `--recursive` | [§7.1](07-入力.md#71-cli-形式) |
| 取り込み判定 | `wiki: false` で除外、それ以外は取り込む | [§7.2](07-入力.md#72-取り込み判定) |
| フロントマター | YAML。`title`, `summary`, `tags`, `related`, `aliases`, `wiki`, `fragment` を解釈 | [§7.3](07-入力.md#73-フロントマター仕様) |
| ノート出力 | 入口ページ + 断片ページ群として `fragments/<rel>/` 配下（`<rel>` は拡張子を除いた相対パス）に展開 | [FR-05](08-機能要件.md#fr-05-ノート出力断片化) |
| 断片化の基準 | h2 を第一境界として断片化。h2 セクションが 300 行超 & h3 が 2 個以上なら h3 で再分割（殻ページ化） | [FR-05](08-機能要件.md#fr-05-ノート出力断片化) |
| 行数の数え方 | h2 直後〜次の h2 直前の LF 単位行数。空行・コード・h3 見出し行を含む。h2 見出し行は含まない | [FR-05](08-機能要件.md#53-行数の数え方) |
| 断片化オプトアウト | フロントマター `fragment: false` / h2 が無いノートは入口ページに全文を置く。h3 再分割も発生しない | [FR-05](08-機能要件.md#fr-05-ノート出力断片化) |
| 断片ナビ | h2/h3 断片ページ冒頭に `> Parent / Prev / Next` の 1 行＋水平線。殻ページは Parent のみ。入口ページはナビなし | [FR-05a](08-機能要件.md#fr-05a-断片ナビゲーション) |
| Related の配置 | 入口ページのみに付与。断片・殻には出さない | [FR-11](08-機能要件.md#fr-11-related) |
| Backlinks の並び | 参照元ページ出力相対パスの昇順 | [FR-10](08-機能要件.md#fr-10-バックリンク) |
| `[[wikilink]]` | v1 対応。Obsidian 準拠。`[[Foo]]` は入口、`[[Foo#sec]]` は h2 断片へ。埋め込み `![[...]]` はプレーンリンクに縮退 | [FR-06](08-機能要件.md#fr-06-wikilink-解釈) |
| 解決優先度 | (1) aliases/title 完全一致 → (2) basename 完全一致 → (3) 複数候補時 同一ディレクトリ → (4) 最短パス → (5) 相対パス辞書順 | [FR-06](08-機能要件.md#fr-06-wikilink-解釈) |
| タグ索引 | ネストタグ `a/b` は `tags/a.md` と `tags/a/b.md` の両方に集計。リンク先は入口ページ | [FR-07](08-機能要件.md#fr-07-タグ索引) |
| 本文内 `#tag` | v1 では解釈しない | [FR-07](08-機能要件.md#fr-07-タグ索引) |
| 見出し索引 | h1〜h2 を対象。h1→入口、h2→断片ページ | [FR-08](08-機能要件.md#fr-08-見出し索引) |
| リンク索引 | 断片解像度。被参照は Backlinks 側に集約 | [FR-09](08-機能要件.md#fr-09-リンク索引) |
| Backlinks / Related の配置 | 各断片ページ／入口ページ末尾に `---` で区切って自動付与。Related は入口＋断片で計算粒度が異なる（FR-11 参照） | [FR-05](08-機能要件.md#fr-05-ノート出力断片化), [FR-10](08-機能要件.md#fr-10-バックリンク), [FR-11](08-機能要件.md#fr-11-related) |
| フル生成 | v1 は毎回全体再生成。差分更新は将来拡張 | [FR-03](08-機能要件.md#fr-03-フル生成) |
| 除外ディレクトリ | `.git`, `node_modules`, `dist`, `build`, `target`, 隠しディレクトリ（`.wiki` 除く） | [§7.4](07-入力.md#74-除外対象) |
| 走査上限 | ファイル 1 MiB 超・バイナリ・symlink はスキップ。走査ファイル数 10,000 / 取り込みノート数 5,000 / 深さ 20 で警告 | [§7.5](07-入力.md#75-走査ガード), [FR-14](08-機能要件.md#fr-14-走査上限) |
| 衝突時の出力名 | 断片 slug 衝突時は末尾に `-N` を付与 | [FR-05](08-機能要件.md#fr-05-ノート出力断片化) |
