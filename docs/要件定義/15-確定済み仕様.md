# 15. 確定済み仕様

v1.0 策定時点で判断が固まった項目。詳細は対応する FR / 節を参照。

| 項目 | 結論 | 参照 |
|---|---|---|
| ツール名 | `md-wiki` | [§1.1](01-概要.md#11-名称) |
| 対応入力 | `.md` のみ。コードは対象外 | [§2](02-スコープ.md) |
| 入力形式 | 単一ファイル（既定）/ ディレクトリ + `--recursive` | [§7.1](07-入力.md#71-cli-形式) |
| 取り込み判定 | `wiki: false` で除外、それ以外は取り込む | [§7.2](07-入力.md#72-取り込み判定) |
| フロントマター | YAML。`title`, `summary`, `tags`, `related`, `aliases`, `wiki` を解釈 | [§7.3](07-入力.md#73-フロントマター仕様) |
| `[[wikilink]]` | v1 対応。Obsidian 準拠。埋め込み `![[...]]` はプレーンリンクに縮退 | [FR-06](08-機能要件.md#fr-06-wikilink-解釈) |
| 解決優先度 | (1) aliases/title 完全一致 → (2) basename 完全一致 → (3) 複数候補時 同一ディレクトリ → (4) 最短パス → (5) 相対パス辞書順 | [FR-06](08-機能要件.md#fr-06-wikilink-解釈) |
| タグ索引 | ネストタグ `a/b` は `tags/a.md` と `tags/a/b.md` の両方に集計 | [FR-07](08-機能要件.md#fr-07-タグ索引) |
| 本文内 `#tag` | v1 では解釈しない | [FR-07](08-機能要件.md#fr-07-タグ索引) |
| 見出し索引 | h1〜h2 を対象。h3 以上は将来拡張 | [FR-08](08-機能要件.md#fr-08-見出し索引) |
| リンク索引 | 未解決を含めない。被参照は Backlinks 側に集約 | [FR-09](08-機能要件.md#fr-09-リンク索引) |
| Backlinks / Related の配置 | 各ノート本文末尾に `---` で区切って自動付与 | [FR-05](08-機能要件.md#fr-05-ノート出力), [FR-10](08-機能要件.md#fr-10-バックリンク), [FR-11](08-機能要件.md#fr-11-related) |
| フル生成 | v1 は毎回全体再生成。差分更新は将来拡張 | [FR-03](08-機能要件.md#fr-03-フル生成) |
| 除外ディレクトリ | `.git`, `node_modules`, `dist`, `build`, `target`, 隠しディレクトリ（`.wiki` 除く） | [§7.4](07-入力.md#74-除外対象) |
| 走査上限 | ファイル 1 MiB 超・バイナリ・symlink はスキップ。走査ファイル数 10,000 / 取り込みノート数 5,000 / 深さ 20 で警告 | [§7.5](07-入力.md#75-走査ガード), [FR-14](08-機能要件.md#fr-14-走査上限) |
| 衝突時の出力名 | 末尾に `-N` を付与 | [§10 出力パス規則](10-出力要件.md#出力パス規則) |
