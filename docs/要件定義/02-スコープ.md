# 2. スコープ

## 2.1 In Scope
- CLI 実行
- ローカルディレクトリ解析（コード + 手書き md の双方）
- ファイル走査と選別
- 構造抽出（ディレクトリ、技術スタック、エントリポイント、テスト）
- 手書き md ノートの取り込みと統合配置
- Markdown wiki ツリー生成
- ノード間リンク生成（自動リンク、バックリンク、タグ索引）
- 毎回フル生成（再実行で追加 md を反映）

## 2.2 Out of Scope
- AI / LLM による自然文要約生成
- 差分更新（v1 は毎回フル生成）
- ファイル監視による自動再実行（watch モード）
- IDE 連携
- 自動コード修正
- 実行時解析
- UML / 図生成
- 完全な解析保証
