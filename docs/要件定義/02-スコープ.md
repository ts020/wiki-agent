# 2. スコープ

## 2.1 In Scope
- CLI 実行
- 単一 `.md` ファイルの処理
- `--recursive` オプションによるディレクトリ再帰走査
- フロントマター（YAML）の解釈
- ノートの自動断片化（h2／h3 単位の小ページ化）と入口ページからの導線生成
- `[[wikilink]]` の解決と標準 Markdown リンクへの変換（`#見出し` 付きは該当断片ページへ）
- タグ索引（ネストタグ対応）の生成
- 見出し索引（全ノート横断）の生成
- リンク索引（参照関係一覧）の生成
- バックリンク・関連ノートの自動付与
- 未解決リンクの一覧化
- 毎回フル生成（再実行で追加 md を反映）

## 2.2 Out of Scope
- ソースコード（`.rs` `.ts` `.py` 等）の解析・シンボル抽出
- AI / LLM による自然文要約生成
- 差分更新（v1 は毎回フル生成）
- ファイル監視による自動再実行（watch モード）
- IDE 連携
- 実行時解析
- UML / 図生成
- 全文検索インデックスの生成
- ローカル埋め込みによる類似ノート推薦
- `.md` 以外の形式の取り込み（`.org`, `.rst`, `.txt` 等）
