# 11. 制約

- ローカル実行（サーバ常駐・デーモン化しない）
- 出力は Markdown のみ
- 入力ファイルを変更しない（非破壊）
- 外部 API / ネットワーク接続を行わない
- LLM 呼び出しを行わない
- embedding または vector database に依存しない
- 自然言語の意味的類似検索を行わない
- 原文から未明示の意味、感情、関係性、伏線を推論しない
- schema pack 内の任意コード実行を許可しない
- `context` は生成 wiki を破壊・変更しない
- 対象は `.md` ファイルのみ（`.markdown` 等の別拡張子は v1 非対応）
- 入力は UTF-8 前提
