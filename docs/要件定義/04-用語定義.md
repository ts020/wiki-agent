# 4. 用語定義

- **ノート**: 入力となる 1 個の `.md` ファイル
- **wiki ツリー**: 生成物のディレクトリ全体（ノート本体＋索引群）
- **索引**: タグ・見出し・リンクなど、ノート群を横断的に列挙するページ
- **タグ索引**: フロントマター `tags` を集計したページ群（`tags/<tag>.md`）
- **見出し索引**: 全ノートの見出し（h1〜h2）を集約したページ
- **リンク索引**: ノート間の wikilink 参照関係を一覧化したページ
- **wikilink**: Obsidian 由来の `[[NoteName]]` 記法。出力時に標準 Markdown リンクへ変換する
- **バックリンク**: 当該ノートを参照している他ノートの一覧
- **フロントマター**: `.md` 先頭の `---` で囲まれた YAML ブロック
- **フル生成**: 出力先ディレクトリの内容をすべて再生成する（v1 の既定動作）
- **増築**: 入力ディレクトリに `.md` を追加し、再実行することで wiki が拡張されること
