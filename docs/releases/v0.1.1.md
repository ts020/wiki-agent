# md-wiki v0.1.1

Date: 2026-05-11

Distribution-hardening release. The `md-wiki` binary itself is unchanged from 0.1.0; this release exists to ship a safer install pipeline and one additional platform target.

## What changed

- **New target.** `aarch64-unknown-linux-gnu` tarballs are now published alongside the four targets shipped in 0.1.0.
- **Stronger curl install.** `install.sh` now extracts the expected sha256 from `checksums.txt`, computes the actual hash, and compares them directly. The previous `sha256sum -c` path would have accepted a malformed checksum file because GNU coreutils exits 0 when every line is malformed.
- **Fail-closed verification.** Missing `checksums.txt`, missing entry for the asset, or no `sha256sum`/`shasum` on the host all abort the install. Set `MD_WIKI_SKIP_CHECKSUM=1` to opt out.
- **Hardened archive extraction.** `install.sh` extracts into a dedicated subdirectory and rejects an extracted binary whose path escapes it.
- **Pinned Actions.** All GitHub Actions in `release.yml` and `verify.yml` are pinned to commit SHAs.
- **Release flow documented.** See `docs/RELEASING.md`.

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/ts020/wiki-agent/main/install.sh | sh
```

Pin to this version:

```sh
curl -fsSL https://raw.githubusercontent.com/ts020/wiki-agent/main/install.sh | MD_WIKI_VERSION=0.1.1 sh
```

## Verification

The release contains:

- `md-wiki-aarch64-apple-darwin.tar.gz`
- `md-wiki-aarch64-unknown-linux-gnu.tar.gz`
- `md-wiki-x86_64-apple-darwin.tar.gz`
- `md-wiki-x86_64-pc-windows-msvc.tar.gz`
- `md-wiki-x86_64-unknown-linux-gnu.tar.gz`
- `checksums.txt`

Manual verification:

```sh
curl -fsSLO https://github.com/ts020/wiki-agent/releases/download/v0.1.1/checksums.txt
curl -fsSLO https://github.com/ts020/wiki-agent/releases/download/v0.1.1/md-wiki-<target>.tar.gz
sha256sum -c checksums.txt --ignore-missing
```

## Upgrade

```sh
curl -fsSL https://raw.githubusercontent.com/ts020/wiki-agent/main/install.sh | sh
md-wiki --version
```
