# md-wiki v0.1.0 Release Notes Draft

Date: 2026-05-08

## Purpose

This is the first public OSS release of md-wiki, a Rust CLI that turns local Markdown notes into an offline Markdown wiki for human browsing and tool-use agent exploration.

## Highlights

- Offline Markdown-to-Markdown wiki generation
- Entry pages, h2/h3 fragments, hierarchical `_index.md`, tags, headings, links, backlinks, and unresolved reports
- Obsidian-style `[[wikilink]]` resolution with aliases and heading targets
- `wiki: false` exclusion and `fragment: false` fragmentation opt-out
- Agentic search output with `md_wiki` metadata, agent guide, page catalog, and term index
- Large Markdown handling through the current large path
- crates.io package name `md-wiki-cli`, with the installed binary kept as `md-wiki`
- MIT license

## Known Limits

- Markdown only; source code indexing is out of scope
- No HTML site generation
- No watch mode or incremental regeneration
- No AI, LLM, embedding, external API, or network dependency
- `![[embed]]` is treated as a normal link in v0.1.0

## Verification Before Tagging

```sh
cargo test --locked
scripts/verify.sh
cargo package --locked
cargo package --no-verify --list
```

Confirm that the package list does not include `.context/`, `.agents/`, `.claude/`, or local work history.
